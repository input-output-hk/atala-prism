# Credential Verification Web

## Setup

All proto files (the .proto files and the ones generated by protoc) are in the proto dir, which is inside the src dir.

Make sure before installing grpc-web that you have [protobuf](https://github.com/protocolbuffers/protobuf/blob/master/src/README.md) installed, since the make-plugin used by it requires protobuf. Then install [gprc-web](https://github.com/grpc/grpc-web/blob/master/net/grpc/gateway/examples/echo/tutorial.md).

To compile the .proto files protoc must be used, the script to compile is

`protoc -I=. echo.proto --js_out=import_style=commonjs:. --grpc-web_out=import_style=commonjs,mode=grpcwebtext:.`

The server used for the gRPC testing is the one used in [this grpc-web tutorial](https://github.com/grpc/grpc-web/blob/master/net/grpc/gateway/examples/echo/tutorial.md).

## Running the project

We use [nvm](https://github.com/nvm-sh/nvm) to handle the node versions.

To ensure the node version standardized run `nvm use`, which will set the node version to 10.16.3, if installed, otherwise install it and then run the command again.

Prior to running the app install the dependencies with `npm install`.

Lastly, to run the project use `npm run start` or just `npm start`.
